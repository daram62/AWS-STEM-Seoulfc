# 학교 안내 AI Agent 만들기

학교 안내 문서를 학습한 AI Agent를 만들어 질문에 자동으로 답변하는 지능형 도우미를 구축해봅니다.

**CloudFormation**을 활용해 기본적인 환경 세팅을 빠르게 끝내고, 핵심적인 부분을 **AWS Console**로 구현할 예정입니다.

- **난이도**: Medium 🔔
- **예상 소요 시간**: 40분

⚠️ **모든 실습은 us-west-2 (오레곤) 리전에서 진행합니다.**

---
## 실습 시나리오

서울금융고등학교 소프트웨어학과 2학년 **민재**는 학교 홍보부 동아리 부장을 맡고 있습니다.

"소프트웨어학과에서는 어떤 걸 배우나요?"  
"코딩 동아리 활동은 어떻게 하나요?"  
"학교 체험 프로그램은 언제 신청할 수 있어요?"

입시 시즌만 되면 학교 인스타그램 DM과 댓글이 폭발합니다. 홍보부 선생님 한 분이서 모든 문의에 답변하시느라 항상 바쁘십니다. 민재와 동아리 친구들도 돕고 싶지만, 수업 끝나고 답변하기엔 이미 하루가 지나있곤 합니다.

"답변이 늦어서 우리 학교 지원 안 하면 어떡하지...?"

어느 날, 민재는 **AWS STEM Cloud Day**에서 들었던 Bedrock 강의가 떠올랐습니다.

💡 **"그래, AI Agent한테 학교 자료 학습시켜서 24시간 자동 응답 시스템 만들면 되잖아!"**

민재는 바로 동아리 친구들을 모았습니다. 선생님께 소프트웨어과 자료를 받아 검토하고, AWS 문서를 찾아보며 계획을 세웠습니다.

"우리가 배운 클라우드 기술로 학교를 도울 수 있다니, 이거 완전 프로젝트감인데?"

설렘 반, 걱정 반으로 민재와 친구들은 오늘 **서울금융고 AI 안내 Agent** 만들기에 도전합니다!

## 실습에서 배울 AWS 서비스

#### **Amazon S3 (Simple Storage Service)**
확장성과 내구성이 뛰어난 클라우드 저장소입니다. 파일, 이미지, 동영상, 문서 등 다양한 데이터를 저장할 수 있고, 세밀한 권한 관리와 버전 관리 기능도 제공합니다.
실습에서는 학교 안내 PDF를 안전하게 업로드해서 AI 학습 데이터로 활용합니다.

#### **Amazon Bedrock**
AWS의 생성형 AI 서비스로, OpenAI gpt나 Anthropic Claude 같은 유명 AI 모델을 바로 사용할 수 있습니다. Knowledge Base 기능을 통해 S3에 저장된 문서를 읽고, AI가 문서 내용을 이해하고 질의응답에 활용합니다.
별도의 모델 배포나 서버 관리 없이도 우리만의 맞춤형 AI를 만들 수 있습니다.

#### **AWS Lambda**
서버리스로 코드를 자동 실행해주는 서비스입니다. 사용자가 AI Agent에 질문을 입력하면 Lambda 함수가 이를 받아 Bedrock에 요청하고, AI 응답을 다시 웹으로 돌려줍니다.
코드가 짧은 시간만 실행되고, 사용하지 않을 때는 비용이 발생하지 않아서 효율적입니다.

#### **AWS Amplify**
프론트엔드와 백엔드를 손쉽게 배포하는 서비스입니다. HTML, CSS, JavaScript로 만든 웹 사이트를 클릭 몇 번으로 배포할 수 있고, 배포 즉시 HTTPS 보안 연결이 자동으로 활성화됩니다.
실습 AI Agent UI를 빠르고 안전하게 인터넷에 공개할 수 있습니다.

#### **AWS CloudFormation**
인프라 리소스를 코드로 선언하고, 템플릿 파일 하나로 S3, Lambda, IAM 등 여러 리소스를 자동 생성하는 서비스입니다. 복잡한 수동 설정 없이 팀원 모두가 동일한 환경을 만들 수 있어서 협업과 반복 실습에 효과적입니다.
한 번의 클릭으로 모든 인프라가 자동으로 세팅됩니다.
