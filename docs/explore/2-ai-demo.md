# AI Agent 데모 체험

## 직접 체험해보세요!

엔지니어가 AWS의 서비스를 이용해 미리 만들어둔 **3가지 AI Agent 데모**를 자유롭게 체험해보세요!

각 Agent마다 다른 특징을 가지고 있으니 비교하면서 체험해보세요.

**[https://agents-demo.stem.awskorea.kr](https://agents-demo.stem.awskorea.kr)**
![데모 사이트 메인 화면](/img/agent/demo-site.png)
사이트에 접속하면 3가지 Agent를 선택할 수 있는 메인 화면이 나타납니다.
각 Agent를 클릭해서 자유롭게 대화해보세요!

---

## 3가지 데모 소개

### 1️⃣ 스피드 퀴즈 Agent 🎬

**무엇을 하는 Agent?**
- 영화/드라마 제목을 초성으로 퀴즈 내기
- 단계별 힌트 제공
- 빠른 응답 속도

**이렇게 체험해보세요**

1. Agent 실행
2. 초성 퀴즈 풀기
3. 힌트 받으면서 정답 맞추기
4. 3개 이상 맞춰보기!

---

### 2️⃣ 테토/에겐 Agent 💕

**무엇을 하는 Agent?**
- 4가지 페르소나(테토남, 에겐남, 테토녀, 에겐녀)와 대화
- 감정을 이해하고 공감하는 AI
- 자연스러운 대화 능력

**이렇게 체험해보세요**

1. 마음에 드는 페르소나 선택
2. 일상적인 이야기 나누기
3. 고민 상담해보기
4. AI의 공감 능력 확인하기

---

### 3️⃣ 시험기간 도우미 Agent 📚

**무엇을 하는 Agent?**
- 시험 일정 입력하면 학습 계획 **자동 생성**
- 과목별 우선순위 **자동 계산**
- 구글 캘린더 연동 기능
- 실제로 도움이 되는 실용적 Agent

**이렇게 체험해보세요**

1. 시험 일정 말하기 (예: "월요일 수학, 수요일 영어")
2. AI가 자동으로 만든 학습 계획 확인
3. 수정 요청해보기
4. (선택) 구글 캘린더 연동 테스트

---

## 체험하면서 생각해볼 것

**Q1. 어떤 Agent가 가장 인상적이었나요?**

- 스피드 퀴즈 - 빠르고 재미있어서
- 테토/에겐 - 공감을 잘해줘서  
- 시험기간 도우미 - 실제로 도움이 될 것 같아서

**Q2. 시험기간 도우미 Agent가 다른 Agent와 다른 점은?**

힌트: 단순히 대화만 하나요? 아니면 뭔가를 '행동'하나요?

**Q3. 내가 만들고 싶은 Agent는?**

- 재미있는 게임/퀴즈 Agent
- 감정 공유/상담 Agent
- 실제 문제를 해결하는 실용 Agent
