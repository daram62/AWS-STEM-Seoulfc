## AWS STEM Cloud Day란? 💡

STEM은 **Science, Technology, Engineering, Mathematics**의 약자로, 
기술에 능통한 차세대 학생들이 실제 엔지니어들과 상호 작용하는 과정을 통해 
컴퓨터 과학을 추구하고 현장의 가시성을 얻을 수 있도록 하기 위해 시작되었습니다.

이번 STEM Cloud Day는 서울금융고등학교 소프트웨어과 학생들에게 
클라우드 컴퓨팅의 기초부터 실무 응용까지 체험할 수 있는 기회를 제공하고, 
AWS 엔지니어들과의 멘토링을 통해 미래 IT 진로에 대한 구체적인 비전을 제시합니다.
