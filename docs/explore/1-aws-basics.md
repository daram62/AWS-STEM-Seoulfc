# AWS & Cloud 알아보기

## 클라우드 컴퓨팅이란?

인터넷을 통해 컴퓨팅 리소스(서버, 저장소, 데이터베이스 등)를 제공하는 서비스입니다.

### 쉽게 이해하기

**온프레미스 (On-Premise) = 집을 짓는 것**
- 땅 구매 (비싼 초기 비용)
- 집 설계 및 건축
- 직접 관리/수리
- 안 살아도 유지비 발생

**클라우드 (Cloud) = 호텔에 머무는 것**
- 필요할 때만 체크인
- 관리는 호텔이 알아서
- 머문 기간만큼만 비용 지불
- 필요에 따라 다른 방으로 변경 가능

## AWS 소개

### AWS란?

Amazon Web Services - 아마존이 제공하는 클라우드 컴퓨팅 플랫폼입니다.

### 가장 많이 사용되는 클라우드 서비스

- 전 세계 190개 이상의 국가에서 사용
- 넷플릭스, 삼성, 현대자동차 등 수백만 기업이 선택
- 글로벌 클라우드 시장에서 가장 큰 규모

### 200개 이상의 서비스 제공

- 컴퓨팅, 스토리지, 데이터베이스
- AI/ML, IoT, 보안
- 게임, 미디어, 분석 등

## AI Agent란?

AI Agent는 사용자의 요청을 이해하고 정보를 바탕으로 스스로 판단해 행동하는 지능형 인공지능입니다.

### 일반 챗봇 vs AI Agent

**일반 챗봇**

사용자가 질문하면 미리 준비된 답변이나 학습된 정보를 바탕으로 응답합니다.

예시:
- 사용자: "다음 주 월요일 수학, 수요일 영어 시험이야"
- 챗봇: "시험 준비 화이팅하세요! 일반적으로 시험 3일 전부터 복습하는 것이 좋습니다. 취약한 부분을 집중적으로 공부하고, 충분한 수면을 취하세요."

일반적인 조언은 할 수 있지만, 사용자의 구체적인 상황에 맞춘 행동은 하지 못합니다.

**AI Agent**

사용자의 요청을 이해하고, 필요한 정보를 찾아서, 스스로 판단하여 행동합니다.

예시:
- 사용자: "다음 주 월요일 수학, 수요일 영어 시험이야"
- AI Agent의 행동:
  1. "학습 계획을 만들어야겠다" (스스로 판단)
  2. 날짜 계산 및 우선순위 분석 (도구 사용)
  3. 과목별 학습 계획 자동 생성 (문제 해결)
  4. 구글 캘린더 연동 제안 (자동화)

특정 정보를 기반으로 스스로 판단하고 행동할 수 있습니다.

### AI Agent의 핵심 능력

**자율적 판단**
- 무엇을 해야 할지 스스로 결정
- 여러 단계의 작업을 순서대로 처리

**도구 사용**
- 필요한 정보를 검색하고 활용
- 외부 API나 데이터베이스 접근

**문맥 이해**
- 이전 대화 내용을 기억
- 사용자의 의도를 파악

## 오늘 실습에서 사용할 AWS 서비스

### AWS CloudFormation

인프라 자동화 서비스

- 코드로 AWS 리소스 생성/관리
- 오늘 용도: 필요한 AWS 리소스 한 번에 생성

### Amazon S3

Simple Storage Service - 클라우드 파일 저장소

- 무제한 용량의 파일 저장 공간
- 오늘 용도: 학교 안내 PDF 파일 업로드

### Amazon Bedrock

생성형 AI 서비스 플랫폼

- Claude, Llama 등 다양한 AI 모델 사용 가능
- 오늘 용도: AI Agent 만들기

### AWS Lambda

서버리스 컴퓨팅 서비스

- 서버 관리 없이 코드 실행
- 오늘 용도: 웹사이트와 AI Agent 연결

### AWS Amplify

웹/모바일 앱 배포 서비스

- GitHub 연동으로 자동 배포
- 오늘 용도: 챗봇 웹사이트 배포
