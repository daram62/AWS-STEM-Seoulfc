# Agent 스크린샷

실습 가이드에 사용될 Bedrock Agent 관련 스크린샷을 저장하는 폴더입니다.

## 필요한 이미지 목록:

- `agent-1.png` - 에이전트 메뉴
- `agent-2.png` - Agent 선택
- `agent-3.png` - 편집 버튼
- `agent-4.png` - 지식 기반 추가 버튼
- `agent-5.png` - 지식 기반 연결 (지침 입력)
- `agent-6.png` - 저장 후 종료 버튼
- `agent-7.png` - 준비 버튼
- `agent-8.png` - Agent 테스트 (질문/답변)
- `agent-9.png` - 별칭 탭
- `agent-10.png` - 별칭 생성 입력
- `agent-11.png` - 별칭 확인 (별칭 ID 표시)

## 스크린샷 촬영 가이드:

1. Agent 콘솔에서 CloudFormation으로 생성된 Agent 클릭
2. 지식 기반 연결 과정 단계별 촬영
3. 별칭 ID가 명확하게 보이도록 촬영
